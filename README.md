# [UW FISH 572](https://github.com/afsc-gap-products/UW-FISH572-coursework) <img src="https://avatars.githubusercontent.com/u/91760178?s=96&amp;v=4" alt="Logo." align="right" width="139" height="139"/>

"Principles and applications of fisheries-independent surveys" is a 4-credit course (FISH 572) that delves into the importance of fisheries-independent surveys as a cornerstone for fisheries stock assessment and ecosystem research.

> See this repository for coursework. The sibling repository <https://github.com/afsc-gap-products/UW-FISH572> only houses the course website.
